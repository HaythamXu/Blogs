## proxy in command line
```bash
set http_proxy=http://192.168.1.2:1080
set https_proxy=https://192.168.1.2:1080

# if need password
set http_proxy_user=jake
set http_proxy_pass=abcd

# cancle proxy
set http_proxy=
set https_proxy=
```
