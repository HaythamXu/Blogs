* 二叉树分类:
    * 满二叉树: 每个节点都2个子节点

    * 平衡二叉树(Balanced Binary Tree):
    * 平衡二叉排序树():
    * 平衡二叉搜索树(Balanced Binary Search Tree):
* 特殊的二叉树
    * 二叉搜索树(Binary Search Tree)
